# QT-TeacherManagementSystem
### 功能介绍
1.注册与登录：系统可以使用游客或管理员两种身份进行登录。若以游客身份进入系统，只可以使用系统的查询功能；若想以管理员身份进入系统，可以使用预先保存在文件中的管理员账号和密码进行登录，或者通过系统进行注册后登录，管理员身份除了基本的查询功能外，还可以对教师信息及其授课信息进行修改。

2.查询教师信息：在表格中显示各个教师的教师编号、姓名、性别、出生年月、职称、最后学历、工资、婚否等信息。除此以外还可以进行特定类型教师信息的显示，如：男性或女性教师的信息列表、某职称的教师信息列表，某年月以后出生的教师信息列表，某年月以前出生的教师信息列表，未婚教师信息列表等。

3.查询授课信息：通过输入某一教师的教师编号，来查询其授课数据集，授课信息包括课程编号、课程名称、课时、教室、授课班级、学分、学期等信息。

4.修改教师信息：管理员可以增加或删除教师。

5.修改授课信息：管理员可以修改某一教师的授课信息：增加现有课程或者新课程以及删除课程。

6.打开其他文件：除了系统默认打开的文件之外，用户可以导入不同的教师信息文件并进行操作。
### 文件与对象描述
1、Teachers类：管理教师信息，包含用于读取教师信息的数据成员以及用于调试时展示信息的成员函数，通过课程编号与Lessons类进行链接。

2、Lessons类：管理课程信息，包含用于读取课程信息的数据成员以及用于调试时展示信息的成员函数。

3、Log类：存储管理员账户，包含用于读取管理员账号及密码的数据成员以及用于调试时展示信息的成员函数。

4、UI文件：包含登陆界面、菜单界面、信息显示界面、信息修改界面等。

5、Resources文件：包含图标、背景图片等资源文件。
